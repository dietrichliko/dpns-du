#  dpns-du

Usage:

    dpns-du dirame ...

    Get the size of one or more directories on the DPM. Relative directory  (no
    leading slash) are relative to /dpm/oeaw.ac.at/homne/cms/store

    The command requires a configured grid interface and a valid proxy
